### Dev Guide
---

##### Installation for development
```bash
$: git clone git@github.com:relution-io/relution-sdk.git

$: cd relution-sdk
 
$: npm i

$: npm link

$: npm run watch
```

##### Test

```bash
$: npm test
```

##### Api reference
```bash
$: npm run api
```
